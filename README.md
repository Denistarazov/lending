# 🎨 Designer Portfolio Landing Page

A sleek, minimal **portfolio landing page** for a UI/UX designer & frontend developer. Clean typography, smooth scroll interactions, and a refined aesthetic — built with pure HTML, CSS, and JavaScript.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

---

## ✨ Features

- 🖋 **Elegant typography** — Fraunces + DM Sans font pairing
- 🎠 **Smooth scroll navigation** — sticky nav with active state tracking
- 📐 **Minimal design** — clean grid layout, generous whitespace
- 📱 **Fully responsive** — adapts to all screen sizes
- 💼 **Portfolio section** — showcase of selected works
- 👤 **About section** — personal bio and skills overview
- 📬 **Contact section** — clear CTA for potential clients
- ⚡ **Zero dependencies** — pure HTML/CSS/JS, no frameworks

## 🚀 Getting Started

No installation needed — just open in a browser:

```bash

open lending/landing_2/index.html

# Option 2: Serve locally
npx serve lending/landing_2
```
